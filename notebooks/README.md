# notebooks/ — Notebook Interattivi

Jupyter notebook per esplorazione dati, demo dei metodi e valutazione.

## Utilità per l'esame

- **EDA**: analisi esplorativa del dataset (classi, statistiche, distribuzione)
- **Demo interattive**: notebook dedicati per ogni metodo con visualizzazioni e metriche
- **Riproducibilità**: ogni notebook può essere rieseguito per verificare i risultati

## File

| Notebook | Contenuto |
|---|---|
| `01_eda.ipynb` | Analisi esplorativa: classi, dimensioni, bilanciamento, immagini corrotte |
| `02_tv.ipynb` | Total Variation: demo, tuning parametri, valutazione su test set |
| `03_unet.ipynb` | UNet: training, validation, evaluation su tutti i noise level |
| `04_diffpir.ipynb` | DiffPIR: sampling, FFT data-fidelity, confronto visivo |
